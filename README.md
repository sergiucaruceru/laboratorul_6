# Laborator 6 - GitHub Pages

## Pași pentru rulare

1. Creează un repository GitHub numit `laborator6`
2. Încarcă toate fișierele din acest proiect
3. Activează GitHub Pages:
   - Settings → Pages
   - Source → GitHub Actions

## Link GitHub Pages

După publicare, link-ul va avea forma:

https://USERNAME.github.io/laborator6/

## Tehnologii folosite
- HTML
- CSS
- GitHub Pages
- GitHub Actions
- Jekyll
